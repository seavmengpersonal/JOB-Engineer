Require from users to confirm email upon registration in AppThemes themes. 

== Description ==

This plugin extends AppThemes themes with email confirmation feature.


= Supported languages =
* English


== Installation ==

1. Upload the folder 'confirm-email' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Logout, and test creating new account on your site.
